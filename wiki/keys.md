## Windows PowerToys

```
## Keep PowerToy running on background to make setting enabled
1. Microsoft Store -> Search [PowerToys]
2. Open PowerToys, on left side open "Keyboard Manager", and enable it
3. Remap a short cust
4. click "+" to add a rule
   Win+ALT+G  -> disable
   Win+G  -> disable
   Win+ALT+W  -> disable
   Win+W  -> disable
```
